Detailed Description

Describes each collision sphere attached to a bone.

To enable collision handling between sphere and/or capsule and hair curves, we define a 'boneSphere', a sphere whose center is attached to one of the bones used for skinning hairs. One can assign radius as well as initial offset with regard to the bind pose of the attached bone.
