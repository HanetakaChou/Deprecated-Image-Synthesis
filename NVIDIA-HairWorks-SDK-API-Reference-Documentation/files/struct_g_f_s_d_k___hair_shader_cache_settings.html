Detailed Description

shader cache settings to create optimized shader per each hair asset.

By giving hints about which features are used, HairWorks can build optimized shader before runtime. Note that once an option is disabled in shader cache, the feature will not work even if it is enabled in instance descriptor in runtime.

See also:
GFSDK_HairSDK.AddToShaderCache() to add optimized shader.
